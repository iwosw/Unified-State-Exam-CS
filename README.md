# Unified-State-Exam-CS

Красивый статический сайт со шпаргалками по ЕГЭ по информатике.

Что есть:

- все задания `1-27`
- типовые идеи решения
- короткие шаблоны на `Python`, `Java`, `C++` и `Rust`
- готовый деплой на `GitHub Pages` через GitHub Actions

Локальный запуск:

1. Открой `index.html` в браузере.

Публикация:

1. Запушь изменения в `main`.
2. В настройках репозитория открой `Pages`.
3. Выбери `GitHub Actions` как источник деплоя.
